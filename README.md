# vertices_selection_sets
>Blender addon to quickly create and use vertex sets for selection.

## setup:
console commands:
~~~
cd ~/.config/blender/2.79/scripts/addons
git clone https://github.com/volodya-renderberg/vertices_selection_sets.git
~~~
